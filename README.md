# juapache.github.io
Personal site and portfolio landing page for AI governance, peace innovation, and technodiversity work.

## Structure
- `/index.html` – main landing page with research, data lab, writing, teaching, and contact sections.
- `/style.css` – site styles.
- `/pdfs/` – upload course handouts, policy briefs, and lecture decks (update links in `index.html`).
- `/assets/figures/` – exported figures/plots used on the site.
- `/assets/images/` – portraits, logos, and illustrations.
- `/LICENSE` – Creative Commons Attribution-NonCommercial 4.0 International (CC BY-NC 4.0).
